ACCOUNTING SYSTEM

📖 Description
Built a web-based accounting system to manage financial transactions and debt tracking.

🚀 Technologies
- HTML, CSS, JS
- Bootstrap5
- PHP
- MySQL

✨ Features
- Manage receipts and payments.
- Track supplier debts and due payments.
- Export invoices.
- Monitor customer debts and outstanding balances.
